# Soporte

## Cómo presentar issues y obtener ayuda

Este proyecto utiliza los issues de GitHub para rastrear errores y solicitudes de funciones. Por favor, busca los issues existentes antes de abrir nuevos issues para evitar duplicados. Para nuevos issues, presenta tu error o solicitud de función como un nuevo issue.

Para ayuda o preguntas sobre el uso de este proyecto, por favor abre un issue.

## Política de Soporte de GitHub

El soporte para este proyecto se limita a los recursos listados arriba.
